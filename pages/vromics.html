Visualisation and Analysis of Spatiallr Resolved Transcriptomics data.

VR-Omics

Abstract

Spatially resolved transcriptomics is an emerging class of high-throughput technologies that enable biologists to systematically investigate spatiotemporal gene expression patterns. Upon data acquisition, one major hurdle is the subsequent interpretation and visualization of the datasets acquired. To address this challenge, we present a novel data visualization system with interactive functionalities in virtual reality, using an embedded automated workflow for analysis and data processing. The application is designed to help biologists to analyse and interpret spatially resolved transcriptomic datasets. Supported are various technologies such as Visium and Xenium by 10X Genomics, STOmics by BGI, MERFISH by VIZGEN or Tomo-seq. The system is implemented as a local 2D desktop application or as an immersive environment using head-mounted display virtual reality. This allows biologists to interact with the data in novel ways not previously possible, which includes visually exploring the gene expression patterns of an organ and comparing genes based on their 3D expression profiles.

Setup

Automated Workflow

Visium

Download:
1 Download dataset without filter or spatial analysis VISIUM > download > select from drop down menu > leave all other selections blank and select either Process to download the data or Process and Visualise to download the data a directly visualise the data in the VR-Omics visualiser. The dropdown contains all publicly published datasets from https://www.10xgenomics.com/resources/datasets and will be automatically updated if new datasets are being published.
2 Download and filter dataset [can be combined with spatial analysis and output plots] VISIUM > download> select from drop down menu > set filter parameters (more information about filter parameters can be found under “Filter Visium data”. Click Process to download and filter the dataset or Process and Visualise to download and filter the dataset and continue to the VR-Omics visualiser. All filter steps that are left blank will be skipped, only filter steps containing a value will be processed. The downloaded data will be saved at DATAPATH The data is filtered using the embedded Python environment and Scanpy (https://scanpy-tutorials.readthedocs.io/en/latest/spatial/basic-analysis.html), for more information go to “Filter Visium data”.
3 Download and spatially analyse dataset [can be combined with filter and output plots] VISIUM > download > tick the Spatially variable genes box and click Process or Process and Visualise The dataset will be downloaded, and a spatial variable gene analysis will be performed. This step usually takes around 30 minutes depending on the machine and the size of the dataset. Spatially variable genes analysis is performed using the Python package “SpatialDE” https://www.nature.com/articles/nmeth.4636
4 Download data and include output plots [can be combined with filter and spatial analysis] VISIUM > download > tick the “Create output plots” box and select “Process” or “Process and Visualise” The automated workflow will produce a variety of output plots including: - H&E stain image - Spatial cluster - Spatial gene counts - Histogram total counts - UMAPs The plots can be found under DATAPATH/figures
